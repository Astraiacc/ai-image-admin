系统说明
本系统集成百度AI开放平台，支持以下能力：
· 通用物体和场景识别 — 识别图片中的物体、场景类别和标签
· OCR文字识别 — 提取图片中的中英文文字内容
· 图像主体检测 — 定位图片中的主体对象
上传图片后系统将自动调用AI接口进行分析，结果会实时更新展示。

📋 配置说明
1. 申请百度AI接口
访问 https://ai.baidu.com/ → 控制台 → 创建应用
选择"图像识别"服务，获取 API Key 和 Secret Key

2. 配置后端环境
在后端目录创建 .env 文件，填入：
BAIDU_API_KEY=你的APIKey
BAIDU_SECRET_KEY=你的SecretKey
3. 启动后端
cd backend
npm install
npm start
